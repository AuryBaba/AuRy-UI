# AuRy-UI
AuRy-UI

AuRy-UI, Windows üzerinde çalışan, Kali Linux terminal tarzı bir Python terminal uygulamasıdır. Tek pencere, siyah/mavi renk teması ve gelişmiş özelliklerle kullanıcıya gerçekçi bir terminal deneyimi sunar.

Özellikler

Windows CMD komutlarını çalıştırabilir.

Admin kullanıcı ise prompt root㉿KullanıcıAdı olarak gösterilir, normal kullanıcı ise user㉿KullanıcıAdı.

Ctrl+C ile işlemleri durdurabilir, Ctrl+V ile yapıştırma yapabilirsiniz.

Ctrl+L ile ekran tamamen temizlenir.

Sistem çıktıları (komut sonucu) yanlışlıkla silinmez.

Frameless pencere ve özel başlık çubuğu ile terminal hissi arttırıldı.

Tek pencere terminal: Sekme yok, stabil ve büyük çıktılarda dayanıklı.

Bu uygulama tamamen açık kaynak Python kodudur. 
Virüs uyarısı; exe dosyasının nasıl derlendiğinden (PyInstaller) kaynaklanan yaygın bir false positive’tir. 
Kodun tamamı GitHub’da açıktır, inceleyebilirsiniz. 
Microsoft Defender, Kaspersky, ESET, Bitdefender dahil 67+ antivirüs temiz dedi.

English Description


AuRy-UI

AuRy-UI is a Python-based terminal application for Windows, styled like the Kali Linux terminal. It provides a realistic terminal experience with a single window, black/blue theme, and advanced features.

Features

Supports all Windows CMD commands.

Admin users see the prompt as root㉿Username, while normal users see user㉿Username.

Ctrl+C stops running processes, Ctrl+V pastes text.

Ctrl+L clears the screen entirely.

System outputs are protected from accidental deletion.

Frameless window with custom title bar for enhanced terminal feel.

Single-window terminal: No tabs, stable even with large outputs.

This application is a completely open-source Python project.
Virus warning: Any alerts from the EXE are a common false positive caused by how it was compiled (PyInstaller).
The full source code is available on GitHub for review.
It has been confirmed clean by 67+ antivirus programs, including Microsoft Defender, Kaspersky, ESET, and Bitdefender.

# AuRy-UI
AuRy-UI

AuRy-UI, Windows üzerinde çalışan, Kali Linux terminal tarzı bir Python terminal uygulamasıdır. Tek pencere, siyah/mavi renk teması ve gelişmiş özelliklerle kullanıcıya gerçekçi bir terminal deneyimi sunar.

Özellikler

Windows CMD komutlarını çalıştırabilir.

Admin kullanıcı ise prompt root㉿KullanıcıAdı olarak gösterilir, normal kullanıcı ise user㉿KullanıcıAdı.

Ctrl+C ile işlemleri durdurabilir, Ctrl+V ile yapıştırma yapabilirsiniz.

Ctrl+L ile ekran tamamen temizlenir.

Sistem çıktıları (komut sonucu) yanlışlıkla silinmez.

Frameless pencere ve özel başlık çubuğu ile terminal hissi arttırıldı.

Tek pencere terminal: Sekme yok, stabil ve büyük çıktılarda dayanıklı.
